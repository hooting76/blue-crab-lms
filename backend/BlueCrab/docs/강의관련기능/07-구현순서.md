## 1. 프로젝트 준비 단계

### **1.1 환경 설정 및 계획 수립**
- **기간**: 1주
- **담당**: 개발팀 전체
- **작업 내용**:
  - 개발 환경 구축 (JDK, Maven, IDE)
  - Git 저장소 및 브랜치 전략 수립
  - 데이터베이스 설계 및 ERD 완성
  - API 명세서 작성
  - 프로젝트 구조 설계

### **1.2 기본 프로젝트 구조 생성**
- **기간**: 3일
- **담당**: 백엔드 개발자
- **작업 내용**:
  - Spring Boot 프로젝트 생성
  - JPA 엔티티 클래스 생성
  - 기본 Repository, Service, Controller 구조 설계
  - 보안 설정 (JWT, 권한)
  - 기본 예외 처리 및 응답 포맷 설정
