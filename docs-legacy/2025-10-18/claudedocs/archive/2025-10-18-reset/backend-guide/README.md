**백엔드 폴더 구조 설명**

```
backend/BlueCrab/src/main/java/com/bluecrab/
├── config/        # 설정
├── controller/    # REST API 컨트롤러
├── service/       # 비즈니스 로직
├── repository/    # 데이터 접근
├── entity/        # JPA 엔티티
├── dto/           # 데이터 전송 객체
├── security/      # 인증/인가
└── util/          # 유틸리티
```

1. **인덱스 문서** 읽기

4. **API 문서** 참조

## 🎯 활용 팁

- **API 연동 시**: 컨트롤러 분석 → DTO 확인 → 테스트
- **에러 해결 시**: 코드 비교 보고서 → 실제 코드 확인
- **신규 기능 개발**: 폴더 구조 → 레이어별 구현 → 테스트
